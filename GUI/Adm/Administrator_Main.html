Tips and Tricks
  • The database backup function provides you with backup of user data,order data,messages data and so forth based on your choice.
  • All backup files do not contain any template file or accessories.



Tips and Tricks
  • The database recovery function can recovery some or all data in the database
  • All changes not saved before recovery will be discarded



Tips and Tricks
  • The database upgrade function can upgrade the database to a new vision or rollback to a old one.
  • All changes not saved before upgrade or rollback will be discarded



Tips and Tricks
  • The database optimization function can help you to optimize the database and rollback to the original one.
  • All changes not saved before upgrade or rollback will be discarded.
